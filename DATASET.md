# DIV2K Dataset

## Description
The DIV2K dataset is a high-quality (2K resolution) image dataset for super-resolution benchmarking. It contains 800 training images, 100 validation images, and 100 test images.

## License
Please refer to the [official website](https://data.vision.ee.ethz.ch/cvl/DIV2K/) for specific license terms. Generally available for academic and research purposes.

## Source
- [Official Website](https://data.vision.ee.ethz.ch/cvl/DIV2K/)
- [Kaggle Dataset](https://www.kaggle.com/datasets/eesungkim/div2k-dataset)

## Setup
The notebook is configured to run on Kaggle. To run locally:
1.  Download the dataset from Kaggle or the official site.
2.  Update the `DATA_DIR_HR` paths in the notebook to point to your local data directory.
